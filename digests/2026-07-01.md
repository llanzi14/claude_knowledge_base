# Digest — 2026-07-01

Covering June 30 – July 1. Biggest cycle in weeks: **Claude Sonnet 5** shipped and is now the default Claude Code model, and the **Fable 5 / Mythos 5 export-control suspension was lifted** — reversing the June 12 shutdown. Plus Claude Science launch, Microsoft Foundry GA, and a security fix in Claude Code v2.1.196.

## Models

- **Claude Sonnet 5 launched — now default in Claude Code** — native 1M-token context window, promotional pricing $2/$10 per Mtok (input/output) through August 31. Requires Claude Code v2.1.197+. [Anthropic](https://www.anthropic.com/news/claude-sonnet-5) / [Changelog](https://code.claude.com/docs/en/changelog)
- **Fable 5 & Mythos 5 export controls lifted — access restoring** — Department of Commerce lifted the June 12 restriction. Fable 5 available globally from July 1 across Claude Platform, claude.ai, Claude Code, and Cowork. Pro/Max/Team/select Enterprise: included for up to 50% of weekly usage limits through July 7, then usage-credit billing. Mythos 5 restored for a set of US orgs (government approval June 26). Cloud availability (AWS/GCP/Microsoft Foundry) rolling out. `[ACTION]` Prior guidance to pin production to Opus 4.8 only is now stale — see updated Model Strategy section. [Anthropic](https://www.anthropic.com/news/redeploying-fable-5) / [CNBC](https://www.cnbc.com/2026/06/30/anthropic-says-trump-admin-has-lifted-export-controls-on-claude-fable-5-and-mythos-5.html)

## Claude Code

- **v2.1.196** (June 29) — org-default models (`/model` shows "Org default"); readable session names; clickable file attachments; **security fix**: `claude mcp list/get` no longer auto-spawns `.mcp.json` servers from a self-approved-but-untrusted workspace; several background-job/session-status/PowerShell/plugin-validate bug fixes; `/code-review` merged five cleanup finders into one, ~25% token reduction; streaming idle watchdog on by default (aborts+retries after 5 min silent stream, disable via `CLAUDE_ENABLE_STREAM_WATCHDOG=0`). [Changelog](https://code.claude.com/docs/en/changelog)
- **v2.1.197** (June 30) — ships Claude Sonnet 5 access as the new default model. [Changelog](https://code.claude.com/docs/en/changelog)
