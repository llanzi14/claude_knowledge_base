## Claude Code

- **v2.1.206** (2026-07-09) — `/cd` gains directory path suggestions (matching `/add-dir`); new `/doctor` check proposes trimming checked-in `CLAUDE.md` files by cutting content Claude could already derive from the codebase — directly relevant to the community-validated "CLAUDE.md under 200 lines" rule already in this KB. `/commit-push-pr` now auto-allows `git push` to the repo's configured push remote (`remote.pushDefault`), not just `origin`. `EnterWorktree` now asks for confirmation before entering a worktree outside `.claude/worktrees/`. **Background agents now upgrade to the new version in the background right after a Claude Code update**, instead of a slow stale-session upgrade on next attach — relevant to scheduled/headless routines like this one. Also: fixed expired login producing a misleading "issue with the selected model" error instead of prompting `/login`; fixed `--resume`/`--continue` not responding to keyboard input on startup; fixed MCP servers configured via `--mcp-config`/`.mcp.json` ignoring a per-server `request_timeout_ms`; improved `/code-review` findings quality on Opus 4.8. [Changelog](https://code.claude.com/docs/en/changelog)

## Platform & Anthropic News

- **Claude Reflect — usage/habits dashboard, beta** (2026-07-09) — new built-in dashboard on Free/Pro/Max (web + desktop) that summarizes activity over 1/3/6/12 months: recurring topics, most active day/peak hour, and how usage maps to Anthropic's "AI Fluency Framework" (delegation, description, discernment, diligence). Requires Memory to be on. Includes wellbeing controls: quiet hours, break-reminder nudges after a set amount of use, and occasional reflection prompts (e.g. "what's one thing you want to keep doing yourself?"). A future update will add time-spent tracking. Anthropic's stated framing (via head of wellbeing policy Ryn Linthicum): built to help people get *more efficient*, not to maximize time-in-app. [MacRumors](https://www.macrumors.com/2026/07/09/anthropic-reflect-claude-tracking/) / [TestingCatalog](https://www.testingcatalog.com/anthropic-adds-usage-reflection-dashboard-to-claude-for-all-users/)
- **Press reaction to Reflect is mixed** — TechCrunch ran a skeptical take framing Reflect as "quietly selling you on AI" despite the stated wellbeing framing; Engadget and Android Authority both drew Spotify-Wrapped/Digital-Wellbeing comparisons. Worth tracking as an early signal on how a habit-tracking AI feature lands with the press before assuming it's purely a wellbeing play. [TechCrunch](https://techcrunch.com/2026/07/09/anthropics-new-claude-feature-is-quietly-selling-you-on-ai/)
