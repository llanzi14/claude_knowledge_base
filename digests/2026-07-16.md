# Digest — 2026-07-16

- **Claude Code v2.1.211 released** (2026-07-15, not caught in yesterday's digest) — security/reliability release. Two security fixes: **permission previews relayed to chat channels (e.g. Slack approval flows) now neutralize bidirectional-override, zero-width, and look-alike quote characters** in tool inputs, closing a path where a malicious tool input could visually alter what an approver sees before clicking approve; and **auto mode can no longer override a `PreToolUse` hook's `ask` decision for unsandboxed Bash** — a hook that says "ask" now always floors the decision at a prompt instead of being silently auto-approved. Also new: `--forward-subagent-text` flag / `CLAUDE_CODE_FORWARD_SUBAGENT_TEXT` env var to include subagent text/thinking in `stream-json` output. Reliability fixes: parallel sessions no longer all log out simultaneously after wake-from-sleep when sharing one credential store; plugin MCP servers now reconnect after an idle web session wakes; Vertex/Bedrock no longer attempt the default Opus model at startup with spurious fallback notices; file-upload validation fixes (DOS device suffixes, trailing dots, multi-hard-link files); various UI/rendering fixes. `[ACTION]` Update to v2.1.211 — the hook-`ask`-flooring fix matters for any Lanzico automation that relies on a `PreToolUse` hook to force confirmation on risky Bash commands (previously auto mode could silently bypass it); the chat-relay spoofing fix matters if any approval flow (e.g. via Claude Tag / Slack) surfaces permission previews to a human approver. [Changelog](https://code.claude.com/docs/en/changelog)
- **Fable 5 free access extended a third time, now through July 19** (announced ~2026-07-12/13, not previously logged — supersedes the 2026-07-09 KB entry which said "through July 12") — Pro/Max/Team/seat-based Enterprise can keep using Fable 5 for up to 50% of weekly limits at no extra cost through July 19, 11:59:59 PM PT; draws from the same weekly pool, no opt-in needed. Reporting frames this as a competitive response to OpenAI's GPT-5.6 Sol and to leaked references to an "Opus 5" inside Cursor. After July 19, usage reverts to prepaid credits at $10/$50 per Mtok (input/output) — same terms as before, timeline just pushed again. Three extensions in five weeks suggests compute-capacity constraints are still the binding factor, not a fixed rollout plan.
