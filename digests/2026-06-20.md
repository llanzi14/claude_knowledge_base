## Claude Code Releases

- **v2.1.183 (June 19)** — Auto mode safety: destructive git commands (`reset --hard`, `checkout -- .`, `clean -fd`, `stash drop`, `commit --amend` on non-agent commits) and infra-destroy commands (`terraform destroy`, `pulumi destroy`, `cdk destroy`) now blocked unless explicitly requested. Added deprecated-model warnings. New `attribution.sessionUrl` setting to omit claude.ai link from commits/PRs. [Changelog](https://code.claude.com/docs/en/changelog)
- **v2.1.181 (June 17)** — Dynamic config syntax: `/config key=value` from the prompt. Bun runtime upgraded to 1.4. `CLAUDE_CLIENT_PRESENCE_FILE` env var for mobile notification control. Subagent panel auto-hides after 30s. [Changelog](https://code.claude.com/docs/en/changelog)
- **v2.1.179 (June 16)** — Mid-stream connection drops now preserve partial responses (connection resilience). Fixed WSL2 mouse scroll regression. [Changelog](https://code.claude.com/docs/en/changelog)
- **v2.1.178 (June 15) — BREAKING** — `TeamCreate`/`TeamDelete` tools removed; every session now has an implicit team and teammates are spawned directly via `Agent(name:...)`. New `Tool(param:value)` permission syntax (e.g., `Agent(model:opus)`). Nested skills in `.claude/skills` subdirectories with closest-wins collision resolution. [Changelog](https://code.claude.com/docs/en/changelog)
