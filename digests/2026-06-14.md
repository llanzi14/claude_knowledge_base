# Digest — 2026-06-14

Coverage: June 12–14, 2026. Three new Claude Code versions shipped on June 12 alone; Fable 5 / Mythos 5 pricing confirmed; Managed Agents scheduling entered public beta; TCS partnership announced.

## Claude Code versions (all June 12)

- **v2.1.174** — `wheelScrollAccelerationEnabled` setting (fullscreen); fixed `/model` picker now shows the model family Default resolves to; [VSCode] `/usage` dialog now shows cache misses, long-context, and per-skill/agent/plugin/MCP cost breakdowns. [Changelog](https://code.claude.com/docs/en/changelog)
- **v2.1.175** — `enforceAvailableModels` managed setting: when enabled, the `availableModels` allowlist also constrains the Default model (falls back to first allowed model). Useful for enterprise model governance. [Changelog](https://code.claude.com/docs/en/changelog)
- **v2.1.176** (current) — Session titles now generated in conversation language; `footerLinksRegexes` setting for regex-matched footer badges; improved Bedrock credential caching (`Expiration`-based, not fixed 1 h); 10+ bug fixes covering Remote Control, tmux/SSH copy, background sessions, Windows network paths, cloud auth timeouts. [Changelog](https://code.claude.com/docs/en/changelog) | [DevelopersIO summary](https://dev.classmethod.jp/en/articles/20260612-cc-updates-v2-1-175/)

## Models

Currently live via Project Glasswing (US government); expanding to biology researchers next. Not publicly available. ## Managed Agents (public beta, June 9–10)

- **Cron scheduling** — assign an agent a cron expression; each trigger starts a new session and completes the task, no scheduler to host. [Anthropic blog](https://claude.com/blog/whats-new-in-claude-managed-agents) | [TechTimes](https://www.techtimes.com/articles/318163/20260610/claude-managed-agents-add-cron-schedules-credential-vaultsanthropic-beta-puts-agents-autopilot.htm)
- **Credential vaults** — store environment variables (API keys, tokens) server-side; agents receive them at runtime without embedding secrets in prompts.
- **Self-hosted sandboxes** (public beta) — run agent execution inside enterprise boundaries.
- **MCP tunnels** (research preview) — private service access inside enterprise perimeter; request access on Claude Platform.
