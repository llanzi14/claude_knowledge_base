## Claude Code

- **v2.1.198** (July 1) — **Claude in Chrome is now generally available**. New **`/dataviz` skill** ships built-in for chart/dashboard design guidance with a runnable color-palette validator. Background agents launched from `claude agents` now **commit, push, and open a draft PR automatically** when they finish code work in a worktree, instead of stopping to ask. The built-in **Explore agent now inherits the main session's model** (capped at Opus) instead of always running on Haiku — better search quality, higher cost. Subagents and context compaction now inherit the session's extended-thinking configuration. Claude Platform on AWS (`anthropicAws`) added as a gateway upstream provider. **`/agents` wizard removed** — create/manage subagents by asking Claude or editing `.claude/agents/` directly. Numerous reliability fixes (network-drop retry with backoff, stuck background tasks, `/diff` panel refresh, AWS STS token auto-refresh, macOS local-network access, worktree `/desktop` errors). `[ACTION]` Update to v2.1.198 for Claude in Chrome + `/dataviz`. [Changelog](https://code.claude.com/docs/en/changelog)
