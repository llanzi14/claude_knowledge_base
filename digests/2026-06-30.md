## Platform (added to KB today — previously untracked)

- **Claude on Microsoft Foundry (Azure) — now GA** — Claude Opus 4.8 and Haiku 4.5 are generally available in Azure via Microsoft Foundry, using existing Azure identity (Entra/IAM), billing, governance, and optional US data zone. Joins Claude Platform on AWS for full multi-cloud coverage. `[ACTION]` Relevant if any Lanzico client work runs on Azure. [Anthropic news](https://www.anthropic.com/news)

- **MCP Connector Observability + in-app directory submission** — Admins and org owners can now monitor adoption, errors, latency, and per-connector token usage across all Claude products. MCP connectors can also be submitted directly from within Claude's in-app directory (no separate developer portal needed). Useful for Lanzico admins managing the Notion, Slack, Gmail, Google Drive, Miro, Canva, and Clay connectors. [Releasebot](https://releasebot.io/updates/anthropic/claude)

- **Trusted Devices for Remote Control Admins** — Team & Enterprise plan admins can now require members to verify their device before being permitted to view or steer a local Claude Code session remotely. Addresses security gaps in remote pair-programming and session-sharing workflows. [Releasebot](https://releasebot.io/updates/anthropic/claude)
