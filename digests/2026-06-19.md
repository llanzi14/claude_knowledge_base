# Digest — 2026-06-19

Coverage: 2026-06-13 through 2026-06-19. Sources: official changelog, Anthropic news, community searches.

## Claude Code (versions 2.1.175 → 2.1.183)

- **2.1.183 (2026-06-19):** Auto mode now blocks destructive git commands (`git commit --amend` on non-agent commits, `terraform destroy`, `pulumi destroy`, `cdk destroy`) unless explicitly requested. Adds `attribution.sessionUrl` setting to omit session links from commits/PRs. Adds `/config --help` for shorthand keys. Fixed MCP server auth exposure in headless mode. [Changelog](https://code.claude.com/docs/en/changelog)
- **2.1.181 (2026-06-17):** `/config key=value` syntax sets any config from the prompt mid-session (works in interactive, `-p`, and Remote Control). Adds `CLAUDE_CLIENT_PRESENCE_FILE` env var for mobile notifications. Improved long-paragraph streaming (line-by-line). Upgraded bundled Bun runtime to 1.4. [Changelog](https://code.claude.com/docs/en/changelog)
- **2.1.179 (2026-06-16):** Polish release — fixed mid-stream connection drops with partial-response preservation, WSL2 mouse-wheel scrolling, sandbox glob on Linux, stacked welcome banners. [Changelog](https://code.claude.com/docs/en/changelog)
- **2.1.178 (2026-06-15):** `TeamCreate`/`TeamDelete` removed; every session now has an implicit team. New `Tool(param:value)` permission syntax matches on tool input parameters. Skills in nested `.claude/skills` directories now load; closest-directory version wins on name clash. Auto mode subagent spawns evaluated by classifier before launch. [Changelog](https://code.claude.com/docs/en/changelog)
- **2.1.175–176 (2026-06-12):** Session titles generated in conversation language (configurable via `language` setting). `enforceAvailableModels` managed setting constrains Default model to an allowlist. Mouse-wheel acceleration toggle (`wheelScrollAccelerationEnabled`). [Changelog](https://code.claude.com/docs/en/changelog)
