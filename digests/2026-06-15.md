## Claude Fable 5 & Mythos 5 — full details

- Model IDs: `claude-fable-5` (GA) and `claude-mythos-5` (Project Glasswing only, requires approval).
- **Pricing:** $10/M input · $50/M output — 2× Opus 4.8. Same pricing for both models.
- **Context:** 1M token context window, up to 128k output tokens per request.
- **Capabilities:** ~80% SWE-Bench Pro; leads agentic-coding evals. Adaptive thinking always on (no way to disable). Raw chain-of-thought is **never** returned; use `thinking.display: "summarized"` for readable reasoning summaries.
- **Safety refusals:** Fable 5 can refuse requests (`stop_reason: "refusal"`, HTTP 200 — not an error). Use the `fallbacks` parameter (API beta) or SDK middleware for automatic retry on another model. Refused requests before any output are **not billed**. [Official docs](https://platform.claude.com/docs/en/about-claude/models/introducing-claude-fable-5-and-claude-mythos-5)
- **Data retention:** 30-day retention; zero-data-retention not available for Fable 5 or Mythos 5.
