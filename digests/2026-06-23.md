# Digest — 2026-06-23

Covers: June 22–23, 2026. One day after the June 22 catch-up.

## Claude Code v2.1.186 (released June 22)

- **`claude mcp login/logout <name>`** — authenticate MCP servers from the CLI without the interactive `/mcp` menu. Useful in scripted or headless environments. [GitHub releases](https://github.com/anthropics/claude-code/releases)
- **`!` bash commands auto-trigger Claude responses** — shell commands prefixed with `!` now produce an immediate assistant reply; disable with `"respondToBashCommands": false` in settings. [GitHub releases](https://github.com/anthropics/claude-code/releases)
- **`/workflows` status filter** — press `f` inside the agent detail view to filter by status. [GitHub releases](https://github.com/anthropics/claude-code/releases)
- **Skills section in `/plugin` Installed tab** — installed skills now visible alongside plugins in `/plugin`. [GitHub releases](https://github.com/anthropics/claude-code/releases)
- **`teammateMode: "iterm2"`** — new setting with an auto-mode warning when using iTerm2 as a teammate. [GitHub releases](https://github.com/anthropics/claude-code/releases)
- **`CLAUDE_CODE_MAX_RETRIES` cap raised to 15** — was previously lower; useful for flaky network environments. [GitHub releases](https://github.com/anthropics/claude-code/releases)
- **`Agent(type)` deny rules now enforce on named subagent spawns** — security fix; previously allowed-types rules could be bypassed when spawning subagents by name. [GitHub releases](https://github.com/anthropics/claude-code/releases)
- **Bug fixes (33 total)**: streaming failures after machine sleep, subagent transcript scroll bleeding, background task previews showing raw tool names, Chrome tab-group isolation for concurrent CLI sessions, Esc/Ctrl+C unresponsiveness during background agents, feature-gated tools slipping through before flags loaded, session cost not showing for Enterprise/Team subscribers. [GitHub releases](https://github.com/anthropics/claude-code/releases)

## Platform & Product

- **Auto mode on third-party providers** — auto mode is now available on Bedrock, Vertex, and Foundry for Opus 4.7 and Opus 4.8, replacing permission prompts with background safety checks. [Claude Code changelog](https://code.claude.com/docs/en/changelog)
- **Claude Design sidebar** — Claude Design now has a dedicated home in the Claude desktop app sidebar; also at claude.ai/design. Includes tighter Claude Code sync, direct canvas editing, and stronger layout controls. [Anthropic](https://www.anthropic.com/news)
- **Claude on Apple Platforms** — Foundation Models framework integration with iOS 27 / macOS 27 / iPadOS 27 / visionOS 27; pass typed outputs from Apple on-device APIs directly into Claude requests. [Anthropic](https://www.anthropic.com/news)
- **Enterprise custom role admin permissions** — custom Enterprise roles can now include admin-level access to billing and privacy controls without requiring full Owner access. [Anthropic release notes](https://platform.claude.com/docs/en/release-notes/overview)
