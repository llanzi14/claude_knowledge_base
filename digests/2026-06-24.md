# Digest — 2026-06-24

Covering June 23–24. Two Claude Code releases, a service outage, and Fable 5 billing now active.

## Claude Code Releases

### v2.1.187 (June 23)
- **`sandbox.credentials` setting** — blocks sandboxed commands from reading credential files and secret env vars. New security control for automated runs. [Changelog](https://code.claude.com/docs/en/changelog)
- **Org model restrictions in all pickers** — `enforceAvailableModels` now enforced in `--model` flag, `/model` command, and `ANTHROPIC_MODEL` env var (not just the UI). Shows "restricted by your organization's settings" on blocked models. [Changelog](https://code.claude.com/docs/en/changelog)
- **Mouse click support in fullscreen select menus** — permission prompts, `/model`, `/config` etc. now respond to mouse clicks in fullscreen mode. [Changelog](https://code.claude.com/docs/en/changelog)
- **Remote MCP tool timeout** — MCP tool calls that hang >5 minutes now abort with an error instead of blocking indefinitely. Override with `CLAUDE_CODE_MCP_TOOL_IDLE_TIMEOUT`. [Changelog](https://code.claude.com/docs/en/changelog)
- **`/btw` navigation** — ←/→ arrow keys now step through earlier `/btw` answers in the overlay. [Changelog](https://code.claude.com/docs/en/changelog)
- **`/install-github-app` simplified** — workflow setup is now optional; you can install just the GitHub App and skip the Actions workflow/secret steps. [Changelog](https://code.claude.com/docs/en/changelog)
- **Background subagents → main session permission prompts** — instead of auto-denying, background agents now surface their permission requests to the main session; dialog shows which agent is asking; Esc denies that one tool only. [Changelog](https://code.claude.com/docs/en/changelog)
- **`/review <pr>` upgraded** — now uses the same engine as `/code-review medium`. [Changelog](https://code.claude.com/docs/en/changelog)
- Bug fixes: remote session startup +2.7s fixed; CJK paste mojibake in byte-event terminals; subagent depth tracking on resume; leaked agent worktree registrations cleaned up; structured output looping fixed. [Changelog](https://code.claude.com/docs/en/changelog)

### v2.1.186 (June 22)
- **`claude mcp login <name>` / `claude mcp logout <name>`** — authenticate MCP servers from CLI without opening the interactive `/mcp` menu. `--no-browser` flag for SSH-based stdin redirect. [Changelog](https://code.claude.com/docs/en/changelog)
- **`/workflows` status filtering** — press `f` in the agent detail view to filter by status. [Changelog](https://code.claude.com/docs/en/changelog)
- **Skills section in `/plugin` Installed tab** — skills now visible alongside plugins in the installed list. [Changelog](https://code.claude.com/docs/en/changelog)
- **`teammateMode: "iterm2"`** — new setting with warning when `it2` CLI is not found. [Changelog](https://code.claude.com/docs/en/changelog)
- **`!` bash commands auto-respond** — Claude now automatically replies to `!<command>` output; disable with `"respondToBashCommands": false` in settings.json. [Changelog](https://code.claude.com/docs/en/changelog)
- **`/review <pr>` uses code-review engine** — same engine as `/code-review medium`. [Changelog](https://code.claude.com/docs/en/changelog)
- **Background subagents in main session** — changed from auto-deny; surfaced to main session with agent attribution. [Changelog](https://code.claude.com/docs/en/changelog)

## Service & Model Status

- **Claude outage — June 23** — elevated error rate across all models, 10:02 AM–12:44 PM ET; now fully resolved. [TechRadar](https://www.techradar.com/news/live/claude-down-june-23-2026) / [Claude Status](https://status.anthropic.com/)
- **Fable 5 & Mythos 5 remain offline** — US government export-control directive (June 12, national security jailbreak concern) suspended both models globally for all users. Free access window also ended June 23, but the billing question is moot while models remain offline. Anthropic disagrees with the directive but is complying. No restoration timeline. [Anthropic statement](https://www.anthropic.com/news/fable-mythos-access) / [Bloomberg](https://www.bloomberg.com/news/articles/2026-06-13/anthropic-says-us-limits-foreign-access-to-fable-5-mythos-5)

## Claude Design (June 17 — previously undocumented in KB)

- Major overhaul: **admin role** for design system approval/locking (enforces brand consistency across AI-generated designs); **direct canvas editing** (drag, resize, align elements); **`/design-sync`** to pull a design system from GitHub into Claude Design from Claude Code; **`/design`** terminal command for full design management from CLI. [VentureBeat](https://venturebeat.com/technology/anthropic-ships-major-claude-design-overhaul-with-design-system-imports-code-round-trips-and-a-fix-for-its-token-burning-problem) / [Anthropic](https://www.anthropic.com/news/claude-design-anthropic-labs)
- Shared usage limits with chat and Claude Code (no separate pool); average turn uses fewer tokens; available on Pro, Max, Team, Enterprise. [TechRepublic](https://www.techrepublic.com/article/news-anthropic-claude-design-overhaul-enterprise-teams/)
