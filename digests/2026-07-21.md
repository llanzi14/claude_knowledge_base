# Digest — 2026-07-21

- **Claude Code v2.1.216 shipped** (2026-07-20) — security/reliability release: fixed workflow saves and **scheduled-task writes following a symlink at `.claude`**, which could redirect writes outside the project (directly relevant to this KB routine, which writes files as a scheduled task); fixed worktree-isolated subagents redirecting git into the shared checkout via `git -C`/`--git-dir`/`GIT_DIR`/`GIT_WORK_TREE` (isolation-bypass fix, continues the 2.1.210 line of fixes); fixed a quadratic message-normalization slowdown causing multi-second stalls in long sessions; fixed auto mode misclassifying commands as denied ("HTTP 401") after an OAuth token expired/rotated mid-session; added `sandbox.filesystem.disabled` setting; AskUserQuestion free-text answers now get neutral wording instead of always "continue." [GitHub CHANGELOG](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md) / [Release](https://github.com/anthropics/claude-code/releases/tag/v2.1.216)
- **Claude Artifacts gain live MCP connector calls** (reported 2026-07-16, added to KB today) — a published artifact can now call a viewer's own MCP connectors each time it's viewed, so a dashboard shows live data / can take actions on demand instead of a static snapshot from the session that built it. Same wave adds public sharing links, editor roles for shared live editing (Team/Enterprise), and artifacts created from Claude Tag sessions. Available on Pro, Max, Team, Enterprise.
