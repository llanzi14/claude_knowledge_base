# Digest — 2026-07-07

One new Claude Code release and a confirmed Enterprise governance feature; one notable but unconfirmed cost-cutting rumor. No new Reddit/community discussion surfaced.

## Claude Code

- **v2.1.202 shipped July 6** — adds a Dynamic Workflow size setting (small/medium/large agent fan-out), new OpenTelemetry workflow attributes, and switches `/review` to a faster single-pass mode (was multi-pass). Also several reliability fixes (Ctrl+R history search, background-session rename, mTLS, Remote Control, voice dictation, skill re-invocation, `/workflows` layout, clearer MCP error messages). [Changelog](https://code.claude.com/docs/en/changelog)

## Platform & Anthropic News

- **Claude Enterprise — admin analytics, model entitlements, spend alerts** (shipped 2026-07-02, reported 2026-07-04) — per-group/per-user cost dashboards, per-role model defaults/entitlements, spend alerts at 75%/90% of org limit, Admin API for scripted governance. Official source. [Claude blog](https://claude.com/blog/giving-admins-more-visibility-and-control-over-claude-usage-and-spend)
- **Unconfirmed: Microsoft reportedly cut Claude Code access over token costs** — aggregator/blog sources (AI Weekly, Enterprise DNA, TheNextWeb, a Forbes contributor piece) claim Microsoft dropped Claude Code for ~5,000 engineers by June 30 due to ~$2,000/engineer/month spend, shifting to GitHub Copilot CLI. No Microsoft or Anthropic statement found; sourcing quality is weak (SEO/aggregator sites, some content reads as speculative). Logged as a flagged rumor, not fact — but directionally consistent with Anthropic's own July 2 spend-alert launch. [AI Weekly](https://aiweekly.co/alerts/microsoft-drops-claude-code-as-enterprise-ai-roi-fails)
